# **App Name**: FarmFriend

## Core Features:

- AI-Powered Crop Doctor: Allow users to upload a photo of a plant and use an AI tool to diagnose the problem, providing a confidence score and solutions categorized into organic and chemical treatments, as well as prevention tips.
- Smart Dashboard: Display hyper-local weather data, key crop market prices with trend indicators, and soil condition indicators (moisture, pH, nitrogen) in a visually appealing card format.
- Knowledge Hub & Community Forum: Provide a simple forum interface where users can post questions and view questions posted by others.
- Integrated AI Chatbot: Integrate an AI chatbot that can answer questions about weather, market prices, crop diseases and treatments, as well as provide guidance on using other app features.
- Language Selection: Allow users to select their preferred language from a dropdown menu (English, Hindi, Telugu, Tamil, Kannada), dynamically updating all UI text, including labels and chatbot responses.

## Style Guidelines:

- Background color: Desaturated earthy green (#E8F5E9).
- Primary color: Vibrant green (#4CAF50) to represent growth and nature.
- Accent color: Warm yellow (#FFEB3B) for highlighting important information and CTAs.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, machined, neutral look. 
- Use universally understood icons (e.g., camera for diagnosis, cloud for weather, market stall for prices). Provide a consistent visual language.
- Mobile-first, responsive layout ensuring seamless adaptation across devices. Use clean lines and plenty of whitespace for readability and user-friendliness.
